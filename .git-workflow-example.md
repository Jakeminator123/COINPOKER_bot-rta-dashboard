# Git Workflow: Multiple AI Agents on Same Task

## Scenario: Fix bug in scanner.py

### Step 1: Create baseline
```bash
git checkout master
git commit -m "Baseline: Current working code"
```

### Step 2: Create branches for 4 agents
```bash
git checkout -b agent-1-fix-scanner
git checkout master
git checkout -b agent-2-fix-scanner
git checkout master
git checkout -b agent-3-fix-scanner
git checkout master
git checkout -b agent-4-fix-scanner
```

### Step 3: Each agent works independently
```bash
# Agent 1 workspace
git checkout agent-1-fix-scanner
# Agent 1 makes changes, commits:
git add scanner.py
git commit -m "Agent 1: Fixed scanner bug using approach A"

# Agent 2 workspace  
git checkout agent-2-fix-scanner
# Agent 2 makes changes, commits:
git add scanner.py
git commit -m "Agent 2: Fixed scanner bug using approach B"

# Agent 3 workspace
git checkout agent-3-fix-scanner
# Agent 3 makes changes, commits:
git add scanner.py
git commit -m "Agent 3: Fixed scanner bug using approach C"

# Agent 4 workspace
git checkout agent-4-fix-scanner
# Agent 4 makes changes, commits:
git add scanner.py
git commit -m "Agent 4: Fixed scanner bug using approach D"
```

### Step 4: Compare solutions
```bash
# See what each agent changed:
git diff master agent-1-fix-scanner
git diff master agent-2-fix-scanner
git diff master agent-3-fix-scanner
git diff master agent-4-fix-scanner

# Visual comparison:
git log --oneline --graph --all
```

### Step 5: Choose best or merge selectively

**Option A: Use best solution**
```bash
git checkout master
git merge agent-2-fix-scanner  # If agent 2 was best
git branch -d agent-1-fix-scanner agent-3-fix-scanner agent-4-fix-scanner
```

**Option B: Merge parts from multiple agents**
```bash
git checkout master
git merge agent-2-fix-scanner  # Start with best overall

# Take specific improvements from others:
git checkout agent-1-fix-scanner -- scanner.py  # Overwrite with agent 1's version
# Review changes, keep what you want
git checkout agent-3-fix-scanner -- utils/helper.py  # Take helper from agent 3
git commit -m "Combined best parts from multiple agents"
```

**Option C: Manual merge**
```bash
git checkout master
git merge agent-2-fix-scanner --no-commit
# Manually edit files to combine best parts
git commit -m "Merged best solutions from all agents"
```

## Advantages

1. **Compare solutions side-by-side**
2. **Test each solution independently**
3. **Combine best parts from multiple agents**
4. **No risk** - can always go back to master
5. **All work preserved** - nothing is lost

## Tips

- Use descriptive branch names: `agent-1-feature-name`
- Commit frequently in each branch
- Test each solution before deciding
- Keep branches until you're sure you don't need them
